# Ksetravid — Official Website

Progressive Death Metal from Bangalore, India.

**Live site:** Built with React 19 + Tailwind CSS 4  
**GitHub:** [saranshshetty2403-cmyk/ksetravid-website](https://github.com/saranshshetty2403-cmyk/ksetravid-website)

## About
Official website for the metal band Ksetravid, featuring:
- Full band bio, discography, and member profiles
- Interactive music player with Spotify embeds
- Video gallery with YouTube integration
- Photo gallery with lightbox
- Tour dates
- Full merch store (8 products from Shopify)
- Press coverage section
- Contact form with booking enquiry

## Tech Stack
- React 19 + TypeScript
- Tailwind CSS 4
- Vite
- Framer Motion
- shadcn/ui
- Wouter (routing)
